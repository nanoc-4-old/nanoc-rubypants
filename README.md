# nanoc-rubypants

This provides a [RubyPants](http://rubydoc.info/gems/rubypants/) filter for [nanoc](http://nanoc.ws).

## Installation

`gem install nanoc-rubypants`

## Usage

```ruby
filter :rubypants
```

This filter takes no options.
